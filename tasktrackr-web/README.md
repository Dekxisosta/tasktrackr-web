# TaskTrackr

## Project structure

```
tasktrackr-web/
├── index.html          Main page markup
├── css/
│   └── style.css        Styling
├── js/
│   └── app.js             Front-end logic (fetch calls, rendering)

│   └── utils.php            Shared helpers
├── data/
│   └── .gitkeep               tasks.json is created at runtime, not committed
├── tests/
│   └── test_tasks.php          Lightweight test script
└── README.md
```
